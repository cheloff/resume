# Eli Gundry's Resume

This is my resume and it's written with
[Pandoc](http://johnmacfarlane.net/pandoc/) in mind. It is written in Markdown
and styled for PDF using LaTeX.

## Usage

To compile a PDF:

```shell
make pdf
```

To compile to LaTeX:

```shell
make latex
```

To compile to a standalone HTML document:

```shell
make html
```
